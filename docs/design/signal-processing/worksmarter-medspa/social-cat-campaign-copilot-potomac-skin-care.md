## 1. What this is

Med spas are told they must be visible on **Instagram, Facebook, Facebook groups, Nextdoor**, and more — but **generating and posting content for all of them is more than a full-time job**.

**Social Cat** does not replace that human. It **compresses** the work into a guided weekly rhythm:

```text
Roam memory + campaign brief + brand voice
        ↓
Social Cat: weekly plan → drafts → checklist → “post this now”
        ↓
Human reviews → publishes as the spa (or schedules via GHL where wired)
        ↓
Cat logs what shipped; suggests next week
```

| What Social Cat is | What Social Cat is not |
|--------------------|-------------------------|
| Step-by-step **campaign coach** | Auto-poster on 12 platforms |
| **Draft factory** (one angle → IG / FB / group reply) | Same spam caption everywhere |
| **Weekly runbook** (~30–60 min/week) | Full-time social manager replacement |
| Disclosed **marketing cat** assisting drafts | Fake neighbor or hidden bot |
| **Organic** community presence | Paid Meta/Nextdoor ads (see consent-machine) |

**Useful because:** Potomac Skin Care’s roam showed strong **Google** trust, active **IG/FB**, competitors with **polished feeds**, and a **VIP + consult-first** story that must stay consistent — without hiring a social team.

## 2. Why not “be on every platform” the old way

| Old advice | Problem for a solo spa |
|------------|------------------------|
| “Be everywhere” | 10+ channels × 3–5 posts/week = unsustainable |
| Hire an agency for $3k/mo | Competes with Work Smarter’s own SKUs if positioned wrong |
| Cross-post identical content | Lazy; groups flag promo; off-brand for affluent trust |
| Bot auto-posts to Nextdoor / FB groups | **ToS + API blocked + STUCK_TRAP** |

Social Cat reframes: **pick 2–4 channels per quarter**, one **campaign theme** per week, **human publishes** with cat-drafted copy.

## 3. Channels (Potomac Skin Care pilot)

| Channel | Social Cat role | Automation |
|---------|-----------------|------------|
| **Instagram** (@potomacskincare) | Feed + Stories drafts; seasonal hooks | Human publish; optional GHL/scheduler later |
| **Facebook Page** | Mirror/adapt IG; event + consult CTAs | Human publish |
| **Facebook Groups** | **Draft** helpful replies when spa is **already a member/admin** | **Human paste only** — no bot join/post |
| **Nextdoor** | **Draft** local neighbor-tone tips **if** staff has account | **Human post manually** — no API in v0 |
| **Google Business Profile** | Occasional update / review-thank-you post draft | Human publish |
| **LinkedIn Company Page** | Optional B2B-light; secondary for this spa | Human publish — **not** LinkedIn LaaS cold DMs |
| **TikTok / X** | Out of pilot scope unless client requests | — |

**Alignment with [consent-machine](./consent-machine-potomac-skin-care.md):** v0 pilot remains **no paid** Meta/LinkedIn/TikTok **ads**. Social Cat covers **organic** presence only.

**Alignment with roam lab:** Nextdoor / private Facebook are **not automated roam sources** ([adopt-a-cat-bot.md §12.2](../../adopt-a-cat-bot.md)) — Social Cat **does not scrape them**; it helps **humans post** when they already have access.

## 4. Weekly “hand holding” workflow

Default: **~45 minutes/week** for one office manager + physician skim on medical lines.

| Day | Step | Cat output | Human action |
|-----|------|------------|--------------|
| **Mon — Plan** (5 min) | Approve weekly theme | 3 channels + 1 campaign hook from roam/brief | Pick “yes / tweak” |
| **Tue — Create** (15 min) | Draft assets | IG caption + FB variant + optional group reply | Edit; physician skim if needed |
| **Wed — Post** (5 min) | Publish checklist | “Post IG + FB now” + disclosure snippet | Tap publish; log URL |
| **Thu — Engage** (10 min) | Engagement prompt | **One** optional group/Nextdoor draft (helpful, not promo) | Post only if relevant thread exists |
| **Fri — Review** (5 min) | Scorecard | “3/3 posted; next week: body contouring consult angle” | Adjust next plan |

```text
┌─────────────────────────────────────────────────────────┐
│  SOCIAL CAT WEEKLY LOOP                                    │
│  Plan → Draft → Approve → Publish (human) → Log → Next   │
└─────────────────────────────────────────────────────────┘
```

## 5. Content from one story (many shapes)

Social Cat reuses **Consent Cat roam** + [campaign brief](./campaign-brief-potomac-skin-care.md) segments:

| Source theme | Instagram | Facebook Page | Group / Nextdoor (careful) |
|--------------|-----------|---------------|----------------------------|
| Physician-trust since 1998 | Carousel: “3 questions before Botox” | Same + consult link | “Local tip: choose physician-led consult” — **no hard sell** |
| VIP re-engagement (M15) | Story: “Feel like a VIP — book consult” | Email cross-promo to **opt-in** list only | ❌ Skip neighbor groups for VIP promo |
| Pre-season (Great Falls segment) | Seasonal skin refresh | Event-ready skin hook | Soft community tone if allowed by group rules |
| Google 5★ / 44 reviews | Thank-you post (no patient IDs) | Share Google link | ❌ Not “review us” spam in groups |

**Rule:** Group/Nextdoor drafts are **educational or neighbor-helpful**, not “book Botox today.”

## 8. STUCK_TRAP (Social Cat edition)

| Trap | Why wrong |
|------|-----------|
| Auto-post to FB groups / Nextdoor | ToS; ban risk; feels like spam |
| Bot joins groups as the spa | Impersonation / policy violation |
| Identical promo on 10 platforms | Lazy; affluent audience rejects |
| “50% off Botox” in neighbor groups | Off-brand; group admin removal |
| Cat posts without human approval | Breaks P1 human-in-the-loop |
| Replace consent SMS/email with social blasts | Social = awareness; nurture = opt-in |
| Pretend to be a neighbor | Monkey business — opposite of Cat Bot ethics |

## 9. Success metrics (client pilot)

| Metric | Target (8-week pilot) |
|--------|------------------------|
| Weekly plan completed | ≥6 of 8 weeks |
| Posts published (human) | ≥2/week on IG + FB page |
| Physician review when medical claims | 100% when injectables/body mentioned |
| Group/Nextdoor drafts used | ≤1/week; only when thread fits |
| Consult mentions / booking clicks | Track if UTM or “how did you hear” |
| Staff time | Self-report ≤1 hr/week |

Not measured by: follower count alone, autopost volume, or group posts without relevance.
