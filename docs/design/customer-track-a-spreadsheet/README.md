## How to open

1. **Excel:** Open each CSV (Data → From Text if needed, UTF-8).
2. **Google Sheets:** File → Import → Upload → select CSV → “Replace spreadsheet” or new sheet per file.
3. Combine into one workbook: import `Sources` as sheet 1, `For customer review` as sheet 2.

## Before the customer meeting

- Leave `customer_verdict` and `customer_why` blank for them to fill.
- Skim every `url` once more (listings change daily).
- Freight is **not** included on most upstream prices unless noted.

## Notes

- Benchmark comparables use [Liquidation.com](https://www.liquidation.com/furniture.html) furniture pallet auctions (resale ceiling, not buy source).
- Margins are **rough** (asking + ~$2k–$4k freight assumed where excluded); customer should validate.
- Row ranks are agent-style pre-sort; customer labels train the real judgment model later.
