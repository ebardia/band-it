# Band It Layer — deck style (10 slides)

## Visual direction
- **Era:** 1950s newspaper + *retro-future as imagined in the 1950s* (atom age, pulp sci-fi, brass gauges, ray-gun optimism) — not modern cyberpunk.
- **Tone:** Whimsical, editorial, intelligent — never cartoonish or slapstick.
- **Site alignment:** Match Band It landing / Daily newspaper UI.

## Typography (PowerPoint fallbacks)
| Role | Web token | PPTX fallback |
|------|-----------|---------------|
| Headlines | Condensed / lead black | Arial Black, Arial Narrow Bold |
| Body | Georgia serif | Georgia |
| Labels / kickers | Mono uppercase | Consolas or Courier New |
| Accent | Electric blue `#0014FF` | Same |
| Neon highlight (sparingly) | `#FFDD00` | Slide titles or rules only |

## Colors
- Paper: `#FFFFFF` / warm `#FAFAFA`
- Ink: `#0A0A0A`
- Muted: `#5C5C5C`
- Rules: 1–2px black or soft `#E0E0E0`
- Optional yellow rule accent (briefing box style)

## Layout per slide
- Masthead-style **kicker** (mono, small caps): slide number or section
- **Headline** (condensed, large)
- **Body** bullets or short paragraphs (serif)
- **Right or bottom:** one engraved-style illustration per slide (sepia, crosshatch, vintage print)

## Image prompts (shared suffix)
> Vintage 1950s newspaper engraving, sepia ink illustration, brass and vacuum-tube retro-future, whimsical but dignified, crosshatch print texture, no modern UI, no photorealistic faces, editorial figure caption style.
